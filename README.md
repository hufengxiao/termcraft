# TermCraft 🏔️

A voxel world in your terminal. Minecraft, but make it TUI.

## Features (planned)

- Procedural terrain generation (noise-based)
- Multiple block types with colors
- First-person terminal view
- Player movement & interaction
- Day/night cycle

## Run

```bash
cargo run
```

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Space | Jump |
| E | Place block |
| Q | Break block |
| 1-5 | Select block type |
| Esc | Quit |
